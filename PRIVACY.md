# Privacy Policy

tldr-busy-brief is a lightweight summarization tool that processes user-provided text to produce structured summaries. This policy explains how data is handled.

## Data Accessed

- Text that users submit to the `explain` tool.

## Data Stored

- No user content is stored by this app.

## External Data Sharing

- The input text is sent to the OpenAI API for processing.

## Data Retention

- This app does not retain user content.
- OpenAIâ€™s API retention policies apply to any requests made.

## Logging

- Operational logs may include timestamps and error messages.
- User content is not persisted in logs.

## Revoking Access

- Remove or rotate your `OPENAI_API_KEY`.
- Disable the app in ChatGPT if you no longer want it to access your text.

## Contact

For privacy questions, open an issue at `https://github.com/iakogwu/tldr-busy-brief`.
